## Heading


If you have an array of strings and want to merge all the elements into a single string, you should use joined().

## Example
~~~

let elements = ["A1", "B2", "C3", "D4"]

let joinedElements = elements.joined()

//Result is "A1B2C3D4"
~~~

Available from iOS 7.0
